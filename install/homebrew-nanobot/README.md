# NanoBot Homebrew Tap

Homebrew tap for NanoBot.Net - A lightweight personal AI assistant framework.

## Usage

```bash
# Add the tap
brew tap mbzcnet/tap

# Install NanoBot
brew install nanobot

# Verify installation
nanobot --version
```

## What is NanoBot?

NanoBot.Net is a lightweight personal AI assistant framework ported from nanobot (Python). It provides:

- Multiple LLM provider support (OpenAI, Azure, Anthropic, etc.)
- Multiple messaging channels (Discord, Slack, Telegram, etc.)
- Extensible tool system
- Memory and context management
- MCP (Model Context Protocol) support
