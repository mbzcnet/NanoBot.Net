# NanoBot 每周更新报告

**日期**: 2025-03-10  
**报告周期**: 2025-03-03 至 2025-03-10  
**项目**: NanoBot.Net (C# 移植版) vs 原版 NanoBot (Python)

## 📊 本周更新概览

### NanoBot.Net (C# 版本) 主要更新

#### 🔧 核心功能增强
- **工具调用可视化**: 在 WebUI 聊天界面新增工具调用可视化功能
- **.NET 10 升级**: 完成从 .NET 8/9 到 .NET 10 的全面升级
- **LLM 配置系统重构**: 简化并优化了 LLM 提供商的配置架构

#### 🎨 WebUI 改进
- **聊天取消功能**: 修复并优化了聊天取消功能的样式和配置
- **多语言支持**: 新增中文本地化支持
- **配置服务**: 实现 WebUI 配置服务，提升用户体验
- **浏览器快照**: 增强浏览器快照功能，改进集成测试

#### 🧪 测试和文档
- **JSON 序列化测试**: 添加了全面的 JSON 序列化测试
- **用户管理 API**: 完善了用户管理 API 文档
- **命令差异文档**: 为 NanoBot 命令添加了详细的差异说明

#### 🗂️ 项目整理
- **移除旧文件**: 清理了旧的代理规则文件 (.trae 和 .windsurf)
- **Temp 目录处理**: 从 git 跟踪中移除 Temp 目录

### 原版 NanoBot (Python 版本) 主要更新

#### 🔧 渠道功能增强
- **QQ 渠道**: 支持使用 markdown payload 发送消息
- **钉钉渠道**: 新增群聊消息支持，修复群回复路由
- **飞书渠道**: 
  - 新增音频转录功能（使用 Groq Whisper）
  - 智能消息格式选择
  - 修复音频文件扩展名兼容性
- **Telegram 渠道**: 
  - 新增群组主题支持
  - 改进流式传输用户体验
  - 新增表格渲染功能
  - 修复代理配置问题
- **Discord 渠道**: 新增文件附件支持
- **WhatsApp 渠道**: 修复媒体消息处理

#### 🏗️ 架构改进
- **多实例支持**: 新增 `--config` 参数支持多实例配置
- **MCP 增强**: 
  - 新增 SSE 传输支持，支持自动检测
  - 改进工具调用取消处理
  - 修复进程崩溃问题
- **阿里云编码计划**: 新增阿里云编码计划 API 支持
- **Azure OpenAI**: 新增 Azure OpenAI 支持

#### 🛠️ 工具和配置
- **工具参数自动转换**: 新增工具参数自动类型转换功能
- **实例路径解析**: 统一实例路径解析逻辑
- **内存管理**: 改进内存工具的警告消息和参数处理
- **取消功能**: 改进各种场景下的取消功能处理

#### 🐛 Bug 修复
- **认证**: 修复 allowlist 绕过问题
- **温度设置**: 修复温度配置问题
- **消息处理**: 修复多个渠道的消息处理问题
- **测试**: 修复多个测试失败问题

## 🔄 版本对比分析

### 架构差异
| 方面 | NanoBot.Net (C#) | 原版 NanoBot (Python) |
|------|------------------|----------------------|
| **语言栈** | C#/.NET 10 | Python 3.8+ |
| **UI 框架** | Blazor WebUI | WebUI (React) |
| **配置管理** | 系统化配置服务 | 基于文件的配置 |
| **多实例支持** | 计划中 | 已实现 |
| **渠道支持** | 逐步移植中 | 全面支持 |
